 Q . How to calculate power of a number using recursion?

    x^n = x * x * x..(n times)..* n
    2^4 = 2 * 2 * 2 * 2 
    x^a * x^b = xa^b 
    x^3 * x^4 = x^3+4 = x^7

    x^n = n * x^n-1

Step 2: Base case - the stopping criterion

    n = 0
    n = 1
